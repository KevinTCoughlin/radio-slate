Why it exists

Simple CLI-first control, tray-friendly playback, and a modern Rust architecture for local Fedora workflows.

Install on Fedora

curl -fsSL https://raw.githubusercontent.com/KevinTCoughlin/radio-slate/main/scripts/install-fedora.sh | bash

What is included

  • CLI and tray entry points
  • Local station library import/export
  • Containerized build path
  • CI, release, and Pages automation
